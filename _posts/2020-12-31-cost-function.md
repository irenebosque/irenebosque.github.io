Also called **loss** or **error**  function and normally denoted as $J$.

$J$ is a function that represents an error. It depends on some parameters $\theta$, therefore we can rewrite it as $J(\theta)$. Logically we want the error to be the smallest as possible so the goal is find which parameters $\theta$ gives you the smallest value of $J$. One way to do this is through the  [[Gradient Descent]] approach.
